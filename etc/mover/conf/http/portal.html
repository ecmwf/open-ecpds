Interface Help

A quick reference for the Data Portal interface.

Light / Dark theme

Click the  /  button in the top-right corner to switch between light and dark mode. Your preference is remembered across sessions.

User menu

Click your username in the top-right corner to open the user menu. From there you can sign out of the portal.

Breadcrumb navigation

The bar below the header shows your current location (e.g. Home › folder › subfolder). Click any segment to jump directly to that level. The button on the right goes up one level.

Browsing files

Click a folder to navigate into it. Click a file to download it directly to your browser.

Search

Type in the Search files… box to filter the current directory listing in real time. The filter is case-insensitive and matches anywhere in the file name.

Page size

The selector next to the search box controls how many entries are shown per page. Use the pagination links at the bottom of the table to move between pages.

Column visibility

The dropdown lets you choose which columns are shown:
• Auto — hides columns automatically on narrow screens.
• All — always shows every column.
• Compact — hides the ID column.
• Custom — pick individual columns with checkboxes.

Sorting

Click any column header to sort the listing. Click again to reverse the order. A third click removes the sort.

Session badge

The badge next to the Traffic button shows the number of active protocol sessions (HTTPS, S3, FTP, SFTP/SCP) for your account vs. the configured maximum. A session is counted from authentication and released when it ends; short-lived operations such as individual S3 API calls open and close quickly, so the counter may fluctuate. The value is always at least 1 while you are logged in. The badge turns yellow when capacity is nearly reached and red when the limit is hit — new connections will be refused until an existing one closes.

Traffic

Click the Traffic button to open a chart showing active connections and upload/download rates for your account over the last 24 hours (times in UTC). Use it to identify quieter periods when the portal has less contention and transfers are likely to be faster.

Access Guide

Click the Access Guide button to open connection instructions for all available protocols (HTTPS, S3, FTP, SFTP/SCP). Each card shows your username pre-filled and provides ready-to-use commands with a copy button.
