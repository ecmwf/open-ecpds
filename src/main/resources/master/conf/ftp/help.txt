Use this FTP server to load and download files from the
following places:

ATTACHMENTS : contacts informations for each Destination
COUNTRIES   : remote Hosts organized by Countries (*)
DESTINATIONS: remote Hosts organized by Destinations (*)
HOSTS       : remote Hosts by Identifier (*)
DATA        : data files on Data Movers

(*) Hosts identifiers can be found through the Web
Interface.

For further information point your browser to:

   https://ecpds.ecmwf.int/
